## CRUD Flow
### User Management
1. **Login**: Only registered users can access the dashboard.
2. **Dashboard**: Displays a list of users.
3. **Edit User**: Click "Edit" to update username or email.
4. **Delete User**: Click "Delete" to remove a user.
### Student Management
1. **Navigate**: Click "Students" in the navbar (visible only for logged-in users).
2. **Add Student**: Fill out student details and submit.
3. **View Students**: See all students in a table.
4. **Edit Student**: Update student details via "Edit" button.
5. **Delete Student**: Remove a student via "Delete" button.
> All student CRUD operations are protected; only authenticated users can perform them
